Setup

Install in minutes

Collar AI runs as a Chrome extension. Follow these steps to get started.

1

Download the Extension

Download the latest release from the GitHub repository. Click the green "Code" button, then "Download ZIP". Unzip to a folder on your machine.

2

Enable Developer Mode

Open chrome://extensions in Chrome. Toggle Developer mode on in the top-right corner of the page.

3

Load the Extension

Click "Load unpacked" and select the dist/ folder from the unzipped download. Collar AI will appear in your extensions list.

4

Configure Your API Key

Click the Collar AI icon in your toolbar. Select your AI provider (Claude or OpenAI), paste your API key, and hit Save settings.
